Project Background

The Toilet Transcripts is my first static website, built as a capstone project for a Front-End Web Development course at General Assembly in the spring of 2017. The multi-page site uses media queries and a hamburger menu for a responsive design, and also features an image carousel, embedded video and audio, CSS tranformations, and Google Fonts. Data is stored locally as JavaScript objects and arrays that are manipulated using JavaScript and jQuery.
